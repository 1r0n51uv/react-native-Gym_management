# B2C Mobile Cross-Platform Gym Application

Connected to Gym_management backed, allows users to login, workout whit round timer, save progress and check them and many features

## Technologies

<ul>
  <li>React-Native</li>
   <li>Firebase</li>
   <li>Various Libraries</li>
  </ul>
